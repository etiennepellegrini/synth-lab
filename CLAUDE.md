## Repository Overview

**synth-course** is an interactive course about sound/music synthesis, geared
towards beginners, and especially made for Etienne who just bought a Korg Volca
keys.
In addition to the course (`synth-course.html`), it also contains a "synth
simulator" (`synth.html`), a guide for the first session with a Volca Keys
(`first-session.md`), and a Volca Keys cheatsheet (`cheat-sheet.md`).
`manifest.json` and `sw.js` were created to make `synth.html` an installable
PWA.

The first draft of all these files was created in Claude's chat interface.
